# 更新日志

## v4.60
修复积分计算时可能弹出 "未知错误 #5" 的问题。

## v4.50
修正了对形如 `y=sinx^2` 从 0 到 2pi 积分结果为0的错误。

## v4.40
增加行列式的计算。已知问题: 还无法求出某些5阶以上行列式的值。

## v4.35
修正了显函数求导器对形如 `y=2arcsinx` 的表达式求导结果为 0 的错误。

## v4.30
新增最小二乘法拟合直线的功能。
修正了显函数求导器对形如 `x^(-2)` 的表达式求导结果遗漏括号的错误。

## v4.25
新增拉格朗日插值: 按提示依次输入节点坐标. 按 Esc 或直接回车结束输入，若输入b则跳回上一个节点。
修正了部分程序响应 Tab 键的顺序混乱的错误。

## v4.20
更换了主面板。
一元方程求解器默认求解精度修改为15。
为方便多项式拟合增加空格输入模式，例如输入 `4 -3 2 1`, 则绘制 `y=4x^3-3x^2+2x+1` 的图像。请注意只有一元n次多项式才能用这种输入方式输入。

## v4.10
新增“显函数求导器”。
修正了科学计算器中部分按钮失效的错误。

## v4.00
新增 `lna` 函数，`lnax` 等价于 `ln(abs(x))`。
增加“复合函数生成向导”。
